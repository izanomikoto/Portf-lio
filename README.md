# Portfólio Izano

Meu portfólio pessoal desenvolvido com Next.js, Tailwind CSS e Framer Motion.

## Projetos

### Shamayin
RPG Fusão Hebraico-Asiático - Sistema RPG de alta fidelidade combinando misticismo hebraico com precisão asiática.

### IZANO
App nativo macOS para aprendizado de japonês com Sistema de Repetição Espaçada (SRS), gamificação XP e interface multilíngue.

### Cromiun Sound
Estação de trabalho musical macOS para treinar modelos de voz personalizados com integração RVC.

## Tecnologias

- Next.js 16
- TypeScript
- Tailwind CSS
- Framer Motion

## Como executar

```bash
npm install
npm run dev
```

## Deploy

O projeto está configurado para deploy automático no GitHub Pages via GitHub Actions.
